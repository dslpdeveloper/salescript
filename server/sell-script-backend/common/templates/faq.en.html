How to add questions to the Scenario?

First, you have to add the first reply of your operator to the conversation scenario. Usually, the first question contains greeting and representation of employee. To do this, click on "Add a question". Your question in rectangular shaped block will appear.

How to add answer option to the Script?

Select needed question of your operator, click "Add option" button, and add the answers. The "Add option" button is displayed if one of the questions is selected.

Why only the manager asks questions, and the client only responds?

It is formal. But in general, is not it more productive to maintain initiative of manager and ask questions?

How to edit the contents of questions and answers blocks?

Select the desired block with the mouse - it becomes highlighted in blue. While it is highlighted, you can edit the content in the right column - in the "Node Properties" box.

How can link the questions and the answers blocks?

Choose a block with the client's answer, and you will be able to connect it with one of the Operator's replies by the clicking on the "Link" button, and then, selecting an appropriate Question block. For your convenience, the questions blocks at this point are highlighted in green. Blocks and connection lines of the Scenario aligned automatically.

How to move into the editor field in the Script Editor?

Use the horizontal and vertical scroll bars and the mouse wheel to scroll the editor field. This is important when you are working with the large-scale scripts. In addition, use the Zoom control in the top right corner of the Editor.

Why are blocks "Start" and "Success" and how to use them?

These blocks are required for the script engine for the editor and viewer to know the beginning and the end - please connect Client's response, which is the purpose of your script to the "Success" block, which was originally created by the Editor. This will create a loop in the script, and you will be automatically returned to the start point of the script to begin with another customer after achieving target of the conversation with the current customer in the View mode.

How to save a script that I've created

Fill in the name of the script in the Editor, and click the "Save".

Tell me, is it possible to give access to my script to another users?

Yes, you can give the script access to other people, see "Rights". Just send them an invitation by e-mail, and set permissions. When they enter the SaleScript, passing the link in the e-mail, they will see the script in the Dashboard.

Tell me, is it possible to export the scenario, and use it offline? What about scripts import?

We don't have these opportunities yet. Our users voted for CRM in last polling. We implement CRM, and then, we will suggest import/export in the next voting to find out what things are most important to be done.
